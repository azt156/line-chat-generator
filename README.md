# LINE 教學對話圖產生器 PRO

這是一個單一 HTML 檔案的小工具，可以產生類 LINE 聊天介面圖片。

## 功能

- 預設角色：小惠、小欣老師
- 左右都會顯示角色名稱
- 可自行上傳左右頭像
- 可自行修改角色名稱
- 可自行上傳聊天室背景
- 可匯出 PNG
- 支援四種尺寸：
  1. 長圖模式（向下延伸）
  2. 1:1 IG 圖卡模式
  3. 3:4 教學海報模式
  4. 9:16 真實手機截圖模式
- 支援兩種風格：
  1. 標準 LINE 彩色
  2. 黑白線稿

## 本機使用方式

1. 解壓縮 ZIP
2. 打開資料夾
3. 點兩下 `index.html`
4. 修改內容後按「更新畫面」
5. 按「匯出 PNG」

## 預設頭像

預設頭像放在：

- `assets/xiaohui.png`
- `assets/teacher.png`

若要更換預設頭像，可以直接替換這兩個檔案，檔名保持不變。
