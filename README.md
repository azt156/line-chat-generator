# LINE 教學對話圖產生器 PRO｜自訂每句時間版

## 新增功能
- 可以逐句自訂時間。
- 沒有指定時間時，會依照「開始時間」與「每句自動間隔分鐘」自動產生時間。
- 原本的角色名稱、頭像、背景、尺寸、黑白線稿、PNG 匯出功能都保留。

## 對話格式

一般格式：
小惠：文字
小欣老師：文字

逐句指定時間：
小惠｜07:30：文字
小欣老師｜07:45：文字

也可以用半形：
小惠|07:30:文字
小欣老師|07:45:文字

## 使用方式
1. 解壓縮 ZIP
2. 點兩下 index.html
3. 修改對話內容
4. 按「更新畫面」或直接查看自動更新
5. 按「匯出 PNG」
